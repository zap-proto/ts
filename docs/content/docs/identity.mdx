ZAP implements W3C DID Core 1.0 specification for node identity and authentication.

## Supported DID Methods

| Method | Description | Use Case |
|--------|-------------|----------|
| `did:lux` | Lux blockchain-anchored DIDs | Production deployments |
| `did:key` | Self-certifying DIDs from keys | Development, testing |
| `did:web` | DNS-based DIDs | Web service integration |

## Parsing DIDs

```typescript
import { parseDid, DidMethod, didUri } from '@hanzo/zap';

// Parse a DID string
const did = parseDid('did:lux:z6MkhaXgBZDvotDkL5257faiztiGiC2QtKLGpbnnEGta2doK');

console.log(did.method);  // DidMethod.Lux
console.log(did.id);      // z6MkhaXgBZDvotDkL5257faiztiGiC2QtKLGpbnnEGta2doK

// Convert back to string
console.log(didUri(did)); // did:lux:z6Mk...
```

## Creating DIDs

### From ML-DSA-65 Public Key

```typescript
import { createDidFromKey, DidMethod, didUri } from '@hanzo/zap';

// 1952-byte ML-DSA-65 public key
const publicKey = new Uint8Array(1952);
// ... populate with actual key bytes

// Create did:key
const didKey = createDidFromKey(publicKey, DidMethod.Key);
console.log(didUri(didKey)); // did:key:z6Mk...

// Create did:lux
const didLux = createDidFromKey(publicKey, DidMethod.Lux);
console.log(didUri(didLux)); // did:lux:z6Mk...
```

### From Web Domain

```typescript
import { createDidFromWeb, didUri } from '@hanzo/zap';

// Simple domain
const did1 = createDidFromWeb('example.com');
console.log(didUri(did1)); // did:web:example.com

// With path
const did2 = createDidFromWeb('example.com', 'users/alice');
console.log(didUri(did2)); // did:web:example.com:users:alice
```

## Node Identity

The `NodeIdentity` class combines a DID with cryptographic capabilities:

```typescript
import { NodeIdentity, generateIdentity, createNodeIdentity, Signer } from '@hanzo/zap';

// Create identity with signer (for signing)
const signer: Signer = {
  publicKey: new Uint8Array(1952), // ML-DSA-65 public key
  sign: async (message) => {
    // Sign with private key
    return new Uint8Array(3293); // ML-DSA-65 signature
  },
  verify: async (message, signature) => {
    // Verify signature
    return true;
  },
};

const identity = generateIdentity(signer);

// Check signing capability
if (identity.canSign()) {
  const message = new TextEncoder().encode('Hello, ZAP!');
  const signature = await identity.sign(message);
  const valid = await identity.verify(message, signature);
}

// Get DID document
const doc = identity.document();

// Get DID URI
const uri = identity.uri(); // did:lux:z6Mk...
```

### Verification-Only Identity

```typescript
import { parseDid, createNodeIdentity } from '@hanzo/zap';

const did = parseDid('did:key:z6MkhaXgBZDvotDkL5257faiztiGiC2QtKLGpbnnEGta2doK');
const publicKey = new Uint8Array(1952);

const identity = createNodeIdentity(did, publicKey);

identity.canSign(); // false - no private key
```

## Stake Management

For consensus participation:

```typescript
import { NodeIdentity, generateIdentity, InMemoryStakeRegistry } from '@hanzo/zap';

const identity = generateIdentity(signer);

// Set stake amount
identity.withStake(BigInt('1000000000000000000')); // 1 LUX

// Reference stake registry
identity.withRegistry('lux:0x1234...');

console.log(identity.stake);         // 1000000000000000000n
console.log(identity.stakeRegistry); // lux:0x1234...
```

### Stake Registry Interface

```typescript
import { StakeRegistry, InMemoryStakeRegistry, Did } from '@hanzo/zap';

// Create registry
const registry = new InMemoryStakeRegistry();

// Set stakes
await registry.setStake(did1, BigInt('1000000'));
await registry.setStake(did2, BigInt('2000000'));

// Query stakes
const stake = await registry.getStake(did1);           // 1000000n
const total = await registry.totalStake();             // 3000000n
const hasSufficient = await registry.hasSufficientStake(did1, BigInt('500000')); // true
const weight = await registry.stakeWeight(did1);       // 0.333...
```

## Types

### Did

```typescript
interface Did {
  method: DidMethod;
  id: string;
}

enum DidMethod {
  Lux = 'lux',
  Key = 'key',
  Web = 'web',
}
```

### VerificationMethod

```typescript
interface VerificationMethod {
  id: string;
  type: VerificationMethodType;
  controller: string;
  publicKeyMultibase?: string;
  publicKeyJwk?: Record<string, unknown>;
  blockchainAccountId?: string;
}

enum VerificationMethodType {
  JsonWebKey2020 = 'JsonWebKey2020',
  Multikey = 'Multikey',
  MlDsa65VerificationKey2024 = 'MlDsa65VerificationKey2024',
}
```

### Service

```typescript
interface Service {
  id: string;
  type: ServiceType;
  serviceEndpoint: string | ServiceEndpoint;
}

enum ServiceType {
  ZapAgent = 'ZapAgent',
  DIDCommMessaging = 'DIDCommMessaging',
  LinkedDomains = 'LinkedDomains',
  CredentialRegistry = 'CredentialRegistry',
}
```

### Signer

```typescript
interface Signer {
  publicKey: Uint8Array;
  sign(message: Uint8Array): Promise<Uint8Array>;
  verify(message: Uint8Array, signature: Uint8Array): Promise<boolean>;
}
```
