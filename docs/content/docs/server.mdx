---
title: Server
description: Server class for exposing tools and resources via MCP
---

The `Server` class runs a ZAP server that exposes tools and resources to clients.

## Import

```typescript
import { Server } from '@hanzo/zap';
```

## Constructor

### `new Server(config?: Partial<Config>)`

Create a new ZAP server with optional configuration.

```typescript
const server = new Server({
  listen: '0.0.0.0',
  port: 9999,
  logLevel: 'info',
});
```

**Configuration Options:**

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `listen` | `string` | `'0.0.0.0'` | Listen address |
| `port` | `number` | `9999` | Listen port |
| `logLevel` | `LogLevel` | `'info'` | Log level (debug, info, warn, error) |
| `tlsCert` | `string` | - | Path to TLS certificate |
| `tlsKey` | `string` | - | Path to TLS private key |
| `maxConnections` | `number` | `1000` | Maximum concurrent connections |
| `connectionTimeout` | `number` | `30000` | Connection timeout (ms) |
| `requestTimeout` | `number` | `60000` | Request timeout (ms) |

## Methods

### `run(): Promise<void>`

Start the server and run indefinitely.

```typescript
const server = new Server({ port: 9999 });
await server.run();
// Server runs until process exits
```

This method blocks and runs the event loop. Use it as the main entry point for server applications.

## Example: Basic Server

```typescript
import { Server } from '@hanzo/zap';

const server = new Server({
  listen: '127.0.0.1',
  port: 9999,
  logLevel: 'debug',
});

console.log('Starting ZAP server...');
await server.run();
```

## Example: TLS Server

```typescript
import { Server } from '@hanzo/zap';

const server = new Server({
  port: 9999,
  tlsCert: '/etc/ssl/certs/server.crt',
  tlsKey: '/etc/ssl/private/server.key',
});

await server.run();
```

## Example: With Environment Config

```typescript
import { Server, loadConfigFromEnv, mergeConfig, DEFAULT_CONFIG } from '@hanzo/zap';

const envConfig = loadConfigFromEnv();
const config = mergeConfig(DEFAULT_CONFIG, envConfig);

const server = new Server(config);
await server.run();
```

Environment variables:

```bash
export ZAP_LISTEN=0.0.0.0
export ZAP_PORT=9999
export ZAP_LOG_LEVEL=info
export ZAP_TLS_CERT=/path/to/cert.pem
export ZAP_TLS_KEY=/path/to/key.pem
export ZAP_MAX_CONNECTIONS=500
```

## Graceful Shutdown

Handle signals for graceful shutdown:

```typescript
import { Server } from '@hanzo/zap';

const server = new Server({ port: 9999 });

process.on('SIGINT', () => {
  console.log('Shutting down...');
  process.exit(0);
});

process.on('SIGTERM', () => {
  console.log('Shutting down...');
  process.exit(0);
});

await server.run();
```

## Config Type

```typescript
interface Config {
  listen: string;
  port: number;
  servers: ServerConfig[];
  logLevel: LogLevel;
  tlsCert?: string;
  tlsKey?: string;
  maxConnections?: number;
  connectionTimeout?: number;
  requestTimeout?: number;
}

type LogLevel = 'debug' | 'info' | 'warn' | 'error';
```
