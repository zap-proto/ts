Node.js environments support all ZAP transport types for maximum flexibility and performance.

## Supported Transports

| Transport | Performance | Use Case |
|-----------|-------------|----------|
| `zap` | Highest | Native ZAP protocol |
| `unix` | High | Same-machine IPC |
| `stdio` | High | Subprocess communication |
| `websocket` | Medium | Cross-platform |
| `http` | Medium | REST-style APIs |

## TCP Connection

```typescript
import { Client } from '@hanzo/zap';

// Native ZAP protocol over TCP
const client = await Client.connect('zap://localhost:9999');

// With TLS
const secureClient = await Client.connect('zaps://gateway.example.com:9999');
```

## Unix Socket

For same-machine communication with minimal overhead:

```typescript
import { Gateway } from '@hanzo/zap';

const gateway = new Gateway({
  servers: [
    {
      name: 'local-db',
      url: 'unix:///var/run/mcp-database.sock',
      transport: 'unix',
    },
  ],
});

await gateway.start();
```

## Stdio Transport

Launch MCP servers as subprocesses:

```typescript
import { Gateway } from '@hanzo/zap';

const gateway = new Gateway({
  servers: [
    // NPM package
    {
      name: 'filesystem',
      url: 'stdio://filesystem',
      transport: 'stdio',
      command: 'npx',
      args: ['-y', '@modelcontextprotocol/server-filesystem', '/data'],
    },
    // Python server
    {
      name: 'python-tools',
      url: 'stdio://python-tools',
      transport: 'stdio',
      command: 'python',
      args: ['-m', 'my_mcp_server'],
      env: {
        PYTHONPATH: '/opt/tools',
      },
    },
    // Local binary
    {
      name: 'custom',
      url: 'stdio://custom',
      transport: 'stdio',
      command: '/usr/local/bin/my-mcp-server',
      args: ['--config', '/etc/mcp/config.json'],
    },
  ],
});

await gateway.start();
```

## Server Example

Run ZAP as a standalone server:

```typescript
#!/usr/bin/env node
import { Server, loadConfigFromEnv, mergeConfig, DEFAULT_CONFIG } from '@hanzo/zap';

const config = mergeConfig(DEFAULT_CONFIG, loadConfigFromEnv(), {
  logLevel: 'info',
});

const server = new Server(config);

console.log(`ZAP server starting on ${config.listen}:${config.port}`);
await server.run();
```

## Gateway Example

Aggregate multiple servers:

```typescript
#!/usr/bin/env node
import { Gateway } from '@hanzo/zap';

const gateway = new Gateway({
  listen: '0.0.0.0',
  port: 9999,
  logLevel: 'info',
  servers: [
    {
      name: 'filesystem',
      url: 'stdio://fs',
      transport: 'stdio',
      command: 'npx',
      args: ['-y', '@modelcontextprotocol/server-filesystem', process.cwd()],
    },
    {
      name: 'memory',
      url: 'stdio://memory',
      transport: 'stdio',
      command: 'npx',
      args: ['-y', '@modelcontextprotocol/server-memory'],
    },
  ],
});

// Graceful shutdown
process.on('SIGINT', async () => {
  console.log('\nShutting down...');
  await gateway.stop();
  process.exit(0);
});

process.on('SIGTERM', async () => {
  await gateway.stop();
  process.exit(0);
});

await gateway.start();
console.log(`Gateway running with ${gateway.listServers().length} servers`);

// Keep process alive
await new Promise(() => {});
```

## CLI Script

```typescript
#!/usr/bin/env node
import { Client } from '@hanzo/zap';

const url = process.argv[2] || 'zap://localhost:9999';
const client = await Client.connect(url);

const tools = await client.listTools();
console.log('Available tools:');
for (const tool of tools) {
  console.log(`  ${tool.name}: ${tool.description}`);
}

const resources = await client.listResources();
console.log('\nAvailable resources:');
for (const resource of resources) {
  console.log(`  ${resource.uri} (${resource.mimeType})`);
}

await client.close();
```

## Performance Tips

### Connection Pooling

Reuse client connections:

```typescript
import { Client } from '@hanzo/zap';

let client: Client | null = null;

async function getClient(): Promise<Client> {
  if (!client) {
    client = await Client.connect('zap://localhost:9999');
  }
  return client;
}

// Use the same connection
const c1 = await getClient();
const c2 = await getClient();
// c1 === c2
```

### Parallel Operations

```typescript
const client = await Client.connect('zap://localhost:9999');

// Parallel tool calls
const results = await Promise.all([
  client.callTool('search', { query: 'typescript' }),
  client.callTool('search', { query: 'javascript' }),
  client.callTool('search', { query: 'rust' }),
]);
```

### Unix Sockets for Local Servers

Unix sockets avoid TCP overhead:

```typescript
// Server listens on Unix socket
const gateway = new Gateway({
  // Listen on Unix socket instead of TCP
  servers: [
    {
      name: 'local',
      url: 'unix:///tmp/zap.sock',
      transport: 'unix',
    },
  ],
});
```

## Systemd Service

```ini
[Unit]
Description=ZAP Gateway
After=network.target

[Service]
Type=simple
User=zap
WorkingDirectory=/opt/zap
ExecStart=/usr/bin/node /opt/zap/gateway.js
Restart=always
Environment=ZAP_PORT=9999
Environment=ZAP_LOG_LEVEL=info

[Install]
WantedBy=multi-user.target
```
