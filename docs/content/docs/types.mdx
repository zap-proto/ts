Complete TypeScript type definitions for the ZAP library.

## Tool Types

### Tool

Represents a callable tool exposed by an MCP server.

```typescript
interface Tool {
  name: string;
  description: string;
  schema: Record<string, unknown>;
}
```

**Fields:**

| Field | Type | Description |
|-------|------|-------------|
| `name` | `string` | Unique tool identifier |
| `description` | `string` | Human-readable description |
| `schema` | `Record<string, unknown>` | JSON Schema for tool arguments |

**Example:**

```typescript
const tool: Tool = {
  name: 'search',
  description: 'Search for documents',
  schema: {
    type: 'object',
    properties: {
      query: { type: 'string' },
      limit: { type: 'number', default: 10 },
    },
    required: ['query'],
  },
};
```

## Resource Types

### Resource

Represents a readable resource exposed by an MCP server.

```typescript
interface Resource {
  uri: string;
  name: string;
  description: string;
  mimeType: string;
}
```

**Fields:**

| Field | Type | Description |
|-------|------|-------------|
| `uri` | `string` | Unique resource URI |
| `name` | `string` | Human-readable name |
| `description` | `string` | Resource description |
| `mimeType` | `string` | MIME type (e.g., `text/markdown`) |

### ResourceContent

Content returned when reading a resource.

```typescript
interface ResourceContent {
  uri: string;
  mimeType: string;
  content: string | Uint8Array;
}
```

**Fields:**

| Field | Type | Description |
|-------|------|-------------|
| `uri` | `string` | Resource URI |
| `mimeType` | `string` | Content MIME type |
| `content` | `string \| Uint8Array` | Text or binary content |

## Prompt Types

### Prompt

Represents a reusable prompt template.

```typescript
interface Prompt {
  name: string;
  description: string;
  arguments: PromptArgument[];
}
```

### PromptArgument

An argument definition for a prompt.

```typescript
interface PromptArgument {
  name: string;
  description: string;
  required: boolean;
}
```

### PromptMessage

A message in a prompt conversation.

```typescript
interface PromptMessage {
  role: 'user' | 'assistant' | 'system';
  content: TextContent | ImageContent | ResourceContent;
}
```

## Content Types

### TextContent

Plain text content.

```typescript
interface TextContent {
  type: 'text';
  text: string;
}
```

### ImageContent

Binary image content.

```typescript
interface ImageContent {
  type: 'image';
  data: Uint8Array;
  mimeType: string;
}
```

## Server Types

### ServerInfo

Information about an MCP server.

```typescript
interface ServerInfo {
  name: string;
  version: string;
  capabilities: ServerCapabilities;
}
```

### ServerCapabilities

Capabilities supported by a server.

```typescript
interface ServerCapabilities {
  tools: boolean;
  resources: boolean;
  prompts: boolean;
  logging: boolean;
}
```

### ConnectedServer

A server connected to the gateway.

```typescript
interface ConnectedServer {
  id: string;
  name: string;
  url: string;
  status: ServerStatus;
  tools: number;
  resources: number;
}
```

### ServerStatus

```typescript
type ServerStatus = 'connecting' | 'connected' | 'disconnected' | 'error';
```

## Configuration Types

### Config

Gateway/server configuration.

```typescript
interface Config {
  listen: string;
  port: number;
  servers: ServerConfig[];
  logLevel: LogLevel;
  tlsCert?: string;
  tlsKey?: string;
  maxConnections?: number;
  connectionTimeout?: number;
  requestTimeout?: number;
}
```

### ServerConfig

Configuration for connecting to an MCP server.

```typescript
interface ServerConfig {
  name: string;
  url: string;
  transport: Transport;
  command?: string;
  args?: string[];
  env?: Record<string, string>;
}
```

### Transport

Supported transport types.

```typescript
type Transport = 'stdio' | 'http' | 'websocket' | 'zap' | 'unix';
```

### LogLevel

```typescript
type LogLevel = 'debug' | 'info' | 'warn' | 'error';
```

const VERSION = '0.2.1';
