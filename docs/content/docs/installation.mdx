---
title: Installation
description: Install ZAP in your project
---

Install `@hanzo/zap` using your preferred package manager.

## npm

```bash
npm install @hanzo/zap
```

## pnpm

```bash
pnpm add @hanzo/zap
```

## bun

```bash
bun add @hanzo/zap
```

## Yarn

```bash
yarn add @hanzo/zap
```

## Requirements

- **Node.js**: 18.0 or higher
- **TypeScript**: 5.0 or higher (recommended)
- **Browser**: Modern browsers with WebSocket support

## TypeScript Configuration

ZAP is written in TypeScript and includes full type definitions. No additional `@types` packages are needed.

Recommended `tsconfig.json` settings:

```json
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "ESNext",
    "moduleResolution": "bundler",
    "strict": true,
    "esModuleInterop": true
  }
}
```

## ESM Only

ZAP is distributed as ESM (ECMAScript Modules) only. Ensure your project supports ESM imports:

```json
{
  "type": "module"
}
```

## Peer Dependencies

ZAP has no required peer dependencies. All ZAP functionality is bundled.

## Optional Dependencies

For post-quantum cryptography (ML-DSA-65, ML-KEM-768), you may need additional native bindings:

```bash
# For full PQ crypto support (when available)
npm install liboqs-node
```

> Note: Web Crypto API (X25519, HKDF) is available in all modern environments.
