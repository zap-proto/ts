title: Client

The `Client` class provides a high-level interface for connecting to ZAP gateways and interacting with MCP servers.

## Import

```typescript
import { Client } from '@hanzo/zap';
```

## Static Methods

### `Client.connect(url: string): Promise<Client>`

Connect to a ZAP gateway.

```typescript
const client = await Client.connect('zap://localhost:9999');
```

**Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `url` | `string` | Gateway URL (supports `zap://`, `wss://`, `ws://`) |

**Returns:** `Promise<Client>` - Connected client instance

**Throws:** `ConnectionError` if connection fails

## Instance Methods

### `listTools(): Promise<Tool[]>`

List all available tools from connected servers.

```typescript
const tools = await client.listTools();

for (const tool of tools) {
  console.log(`${tool.name}: ${tool.description}`);
  console.log('Schema:', tool.schema);
}
```

**Returns:** `Promise<Tool[]>` - Array of available tools

### `callTool(name: string, args: Record<string, unknown>): Promise<unknown>`

Call a tool with the provided arguments.

```typescript
const result = await client.callTool('search', {
  query: 'TypeScript generics',
  limit: 10,
});
```

**Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `name` | `string` | Tool name |
| `args` | `Record<string, unknown>` | Tool arguments matching the schema |

**Returns:** `Promise<unknown>` - Tool execution result

**Throws:**
- `ToolNotFoundError` if tool does not exist
- `InvalidArgumentError` if arguments are invalid
- `ServerError` if tool execution fails

### `listResources(): Promise<Resource[]>`

List all available resources from connected servers.

```typescript
const resources = await client.listResources();

for (const resource of resources) {
  console.log(`${resource.name} (${resource.mimeType})`);
  console.log(`  URI: ${resource.uri}`);
  console.log(`  ${resource.description}`);
}
```

**Returns:** `Promise<Resource[]>` - Array of available resources

### `readResource(uri: string): Promise<ResourceContent>`

Read a resource by URI.

```typescript
const content = await client.readResource('file:///path/to/doc.md');

if (typeof content.content === 'string') {
  console.log('Text content:', content.content);
} else {
  console.log('Binary content:', content.content.length, 'bytes');
}
```

**Parameters:**

| Name | Type | Description |
|------|------|-------------|
| `uri` | `string` | Resource URI |

**Returns:** `Promise<ResourceContent>` - Resource content with metadata

**Throws:** `ResourceNotFoundError` if resource does not exist

### `close(): Promise<void>`

Close the connection and release resources.

```typescript
await client.close();
```

Always call `close()` when done to properly clean up connections.

## Example: Complete Workflow

```typescript
import { Client, ToolNotFoundError, ResourceNotFoundError } from '@hanzo/zap';

async function runAgent() {
  const client = await Client.connect('zap://localhost:9999');

  try {
    // Discover capabilities
    const [tools, resources] = await Promise.all([
      client.listTools(),
      client.listResources(),
    ]);

    console.log(`Connected: ${tools.length} tools, ${resources.length} resources`);

    // Use a search tool if available
    const searchTool = tools.find(t => t.name === 'search');
    if (searchTool) {
      const results = await client.callTool('search', {
        query: 'machine learning',
      });
      console.log('Search results:', results);
    }

    // Read documentation if available
    const docs = resources.filter(r => r.mimeType === 'text/markdown');
    for (const doc of docs.slice(0, 3)) {
      const content = await client.readResource(doc.uri);
      console.log(`${doc.name}:`, content.content);
    }
  } catch (error) {
    if (error instanceof ToolNotFoundError) {
      console.error('Missing tool:', error.details?.toolName);
    } else if (error instanceof ResourceNotFoundError) {
      console.error('Missing resource:', error.details?.uri);
    } else {
      throw error;
    }
  } finally {
    await client.close();
  }
}
```
